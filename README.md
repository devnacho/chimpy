# Chimpy

Chimpy syncs your users from your DB to your respective lists in MailChimp.

## Installation

Add this line to your application's Gemfile:

    gem 'chimpy'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chimpy
